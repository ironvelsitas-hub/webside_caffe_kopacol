🏷️ Generate QR Code untuk Meja Cafe IronColol

📖 Cara Penggunaan:

  1. Setiap QR code di bawah ini untuk nomor meja yang berbeda
  2. Print QR code sesuai dengan nomor meja di cafe Anda
  3. Tempelkan QR code di setiap meja
  4. Customer cukup scan QR code untuk langsung terhubung ke menu dan memesan dari meja mereka
  5. Klik tombol "Print All" untuk mencetak semua QR code

⚠️ Penting: Pastikan server backend berjalan di URL yang benar. Ganti localhost:3000 dengan domain Anda jika sudah di-host.
